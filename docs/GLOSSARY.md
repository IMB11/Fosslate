# Glossary

- **Project**: A project is where all the strings for one app/site/thing live. It has a name, an icon, a source language, and target languages.
- **Language**: A language is just a name and a key. The key can be a normal BCP 47 key like `en-GB`, or a custom string if needed.
- **Source Language**: The source language is the language the original strings are written in.
- **Target Language**: A target language is a language we want translations for.
- **Namespace**: A namespace is a group of strings inside a project. String IDs only need to be unique inside that namespace.
- **String**: A string is the original bit of text. It has an ID and a value.
- **String ID**: A string ID is the key we use to find a string inside a namespace.
- **Translation**: A translation is someone's translated value for a string.
- **Author**: The author is the user who added a translation.
- **Rating**: The rating is the score a translation gets from votes.
- **Approval**: An approval means this translation wins, even if another one has a higher rating.
